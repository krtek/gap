What is GAP?

Ever though how cool would be to use virtualenv and pip with your Python Google AppEngine project?

Well, now you can!

Prerequisites

Assuming that you're a Python developer and already use PIP and virtualenv. Download and install Google App Engine SDK.

Usage

Begin with creating virtualenv for your project. 

mkvirtualenv gap-example
workon gap-example

Then install GAP and create your first project:

pip install gap
gap start-project gap-example
cd gap-example

Your first GAP application is ready! Start it by running 

bin/run

and navigate to: http://localhost:8080/. Voil√†!

What now?

Now you can start hacking (go to src/) as you would do in any other Python project. Just remember to use bin/gip, not pip!

bin/gip install <package>

This script uses pip to install dependencies and then symlinks it to working directory.

You can even create requirements.gip file to store all your dependencies and install it later by running: 

bin/gip install -r requirements.gip

Deployment

Ready to push to production? Then do it!

appcfg update src
